# Intini Digital System — Studio Site

Sito web premium per agenzia/studio di sviluppo web.

## Stack
- React 18 + Vite
- Three.js / React Three Fiber (blob 3D hero)
- Framer Motion (animazioni)
- Tailwind CSS
- Lenis (smooth scroll)
- Lucide React (icone)

## Sezioni
1. **Hero** — Blob 3D liquido, headline animata, stats
2. **Manifesto** — Testo animato scroll-driven + stats
3. **Work** — Portfolio bento grid con mock browser frame
4. **Services** — Bento grid servizi con hover 3D tilt
5. **WhyUs** — Tabella comparativa IDS vs Agenzia vs Freelancer
6. **Testimonials** — Recensioni con star rating
7. **Pricing** — 3 piani con scarcity + tabella comparativa toggle
8. **FAQ** — Accordion 8 domande frequenti
9. **Footer** — Form contatto + link + wordmark

## Features speciali
- Mobile menu hamburger full-screen
- WhatsApp floating button con pulse animation
- Scarcity banner nei prezzi
- Confronto feature toggle nei prezzi
- Preloader con progress bar

## Deploy
Pronto per Vercel (vercel.json incluso).

```bash
npm install
npm run dev
```
